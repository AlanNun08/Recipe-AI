# Documentation Organization

This directory contains all project documentation grouped by topic.

## Structure

- `authentication/`
  - Sign-up/login implementation docs
  - Account/email verification docs
  - Mailjet setup and auth architecture diagrams
- `weekly-recipes/current/`
  - Primary weekly recipes documentation set
  - Fix guides, deployment notes, and implementation references
- `weekly-recipes/root-copies/`
  - Older/alternate copies of weekly recipes docs that were previously stored in the repo root
  - Kept because file contents differ from the `current/` versions
- `ideas/`
  - Scratch notes and planning docs

## Quick Start

- Auth docs index: `docs/authentication/DOCS_INDEX.md`
- Auth summary: `docs/authentication/README_AUTHENTICATION.md`
- Weekly recipes index: `docs/weekly-recipes/current/DOCUMENTATION_INDEX.md`
- Weekly recipes quick reference: `docs/weekly-recipes/current/WEEKLY_RECIPES_QUICK_REF.md`
